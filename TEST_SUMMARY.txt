from the database layer.

2. MANUAL ACCEPTANCE CHECKS
---------------------------
Per the project brief, the following flows were verified manually:

- [x] Login: Roles redirect to correct dashboards.

- [x] Data Integrity: Duplicate enrollments are prevented by DB constraints and Service logic.
- [x] UI Responsiveness: All database operations run on background threads (SwingWorker).
- [x] Cloud Connectivity: SSL handshake with TiDB verified.
